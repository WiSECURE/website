FileAegis™
Secure Cloud Storage System

Groundbreaking enterprise-grade cloud file storage
featuring FIDO2 certified passwordless authentication
and NIST compliant encryption

FileAegis is a next generation file exchange system, built to meet enterprise-grade requirements for file sharing and collab.
The system features trending FIDO certified passwordless authentication, NIST compliant encryption, centralized file management, and one-of-a-kind file collab mechanism. WiSECURE aims to revolutionize file exchange in enterprises with security and efficiency going hand in hand.

Lack of Enterprise-grade File Exchange

Since passwords have led to many security issues, the paradigm has been shifted to crypto-based security. Enterprises also need centralized management to robustly control employees' account and privileges and track file exchange activities.

 
Download the product brief of FileAegis

Features of FileAegis

Centralized File Management

IT operators and users can track file exchange and distribution activities through the dashboards.
The centralization enables the IT managers or security officers to enforce policies, manage access, visualize activities and conduct in-depth analysis.

Certified Passwordless Login

With built-in FIDO2 certified passwordless strong authentication, enterprises do not need to manage passwords and won’t be worried about unauthorization. The mechanism can effectively ward off phishing, impersonation, etc.

Certified Passwordless Login

With built-in FIDO2 certified passwordless strong authentication, enterprises do not need to manage passwords and won’t be worried about unauthorization. The mechanism can effectively ward off phishing, impersonation, etc.

Transparent Encryption

All the files uploaded are automatically encrypted. Even if the file is intercepted or stolen, it will take several decades to decrypt it. Enterprises can securely exchange and distribute confidential files using FileAegis.

Crypto-based Signed Audit Logging

File sharing, exchange and distribution activities are thoroughly recorded. The logs are cryptographically signed by doers’ private keys for accountability. The audit trail is then encrypted. The mechanism allows for enterprise-grade security compliance.

Collab Mechanism

FileAegis supports file archive and revision management. Users can form a group to conduct collaboration. Privileges to view or edit a project can be set to control individual access.

ODM Service

With crypto implementation and ODM service from low-level crypto functions to hardware module development, WiSECURE builds security products from the grounds up and provides ODM service for customers requesting branded security keys.
