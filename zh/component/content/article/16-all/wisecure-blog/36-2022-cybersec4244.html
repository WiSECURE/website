重點參展產品
FileAegis 零信任檔案安全管理平台，滿足企業零信任安全需求 (點此瞭解更多)

✓  獨家專利密碼實作技術，高速加密不影響效能，金鑰儲存不佔空間

✓  一次性連結供外部加密上傳檔案，保護遠距機敏資料傳輸

✓  無密碼身分認證，無需管理任何密碼，降低 IT 管控成本

✓  檔案存取最小特權，符合零信任框架、限縮勒索軟體攻擊

✓  中心化管理檔案流向，滿足稽核及風險控管需求

✓  無密碼登入可使用 SSO 串接企業內部服務，一併導入無密碼身分認證

✓  提供 FileAegis 引擎 API ，彈性整合 AD、ERP、CRM 系統

 

FIDO 企業級方案 (點此瞭解更多)

獨步全球、唯一支援 FIDO 最新規格 (CTAP 2.1) 的 FIDO2 Level 2 安全金鑰，匯智安全科技打造最完善的企業級零信任基礎建設。

✓  支援 EA（Enterprise Attestation），內部安控由企業自行主導，管控認證設備的唯一解方

✓  易用的使用者介面，無需工程人力資源便可發行憑證

✓  100% 抵擋外部不明登入來源

✓  防止釣魚網站攻擊
